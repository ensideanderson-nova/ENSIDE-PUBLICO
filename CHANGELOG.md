# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-11-19

### Adicionado
- Sistema completo de organização com 14 categorias
- Importador universal com detecção inteligente
- Sistema de cores para Finder do macOS
- Visualização HTML interativa com filtros
- Skill personalizada para Claude Code
- Estrutura de 6 bancos (Itaú, Bradesco, Santander, BB, Caixa, Nubank)
- Separação CPF/CNPJ em documentos bancários
- Organização mensal do Financeiro (12 meses)
- Sistema de backup em 3 níveis (Diário/Semanal/Mensal)
- Detecção automática de fraudes e segurança
- Organização de vídeos por categoria
- Sistema de prints de tela com evidências
- Workspace para código e projetos
- Script de instalação automatizado
- Documentação completa (README, Installation, Examples)
- Atalho desktop para importação rápida

### Scripts Incluídos
- `importador_universal.py` - Importador principal
- `organize_master.sh` - Criador da estrutura
- `gerar_html_completo.py` - Gerador de HTML
- `aplicar_cores_completo.sh` - Aplicador de cores
- `file_organizer.py` - Organizador de arquivos
- `pdf_processor.py` - Processador de PDFs
- `general_organizer.py` - Organizador geral

### Funcionalidades
- Detecção inteligente de extratos bancários
- Reconhecimento de notas fiscais
- Identificação de contratos e boletos
- Classificação de documentos de segurança
- Organização automática de código-fonte
- Aplicação de cores por categoria
- Busca em tempo real no HTML
- Filtros por categoria
- Estatísticas do sistema
